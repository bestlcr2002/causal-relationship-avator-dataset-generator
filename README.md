 Causal Relationship Avatar Dataset Generator

A Unity application for generating avatar datasets with causal relationships between DNA parameters.

## Download & Setup

### Clone with Git LFS

**Important:** This repository uses Git LFS for large files. You must install Git LFS before cloning.

```bash
git lfs install
git clone https://github.com/bestlcr2002/causal-relationship-avator-dataset-generator.git
```


## Running the Application

1. Navigate to the downloaded folder
2. Run `UMAdataset.exe`

## Features

### Manual Causal Relationship Creation
- Click any DNA selection button to open a popup list of all 51 DNA elements
- Select source and target DNA parameters
- Choose relationship type: Equal, Minus, Square, SquareRoot, Power, LinearScale, Average, WeightedAverage, Product, GeometricMean
- Optionally add noise with configurable range

### Random Causal Relationship Generation
- Input the desired number of causal relationships to generate
- System randomly generates relationships between DNA elements


## Requirements

- Windows 64-bit
- DirectX 12 compatible graphics card
